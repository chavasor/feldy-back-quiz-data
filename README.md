# Feldy back-relaxation reader poll (May–June 2026)

Informal on-site reader poll of 88 visitors to [feldy.me](https://feldy.me) — an online Feldenkrais Method gentle-movement program — who arrived looking for help with lower back pain. Only 18 percent reported that their back actually relaxed when they tried to relax it. 82 percent described it as held, undifferentiated, or unclear.

This is a small, self-selected sample. Descriptive only.
